# Nexera Features

Nexera is an ERD (Entity-Relationship Diagram) web-based editor with drag and drop functionality for designing database schemas visually.

## Core Features

### Database Management
- **Multiple Databases** - Create and manage multiple database projects
- **Database Dashboard** - View all your databases in a clean card-based grid layout
- **Local Storage** - All data is saved locally in your browser's local storage
- **Auto-save** - Manually save your work at any time with the save button
- **Export Diagrams** - Export your database diagrams for documentation or sharing

### Visual ER Modeling
- **Drag & Drop Interface** - Intuitive drag and drop functionality for tables and elements
- **Table Management** - Add, edit, and delete database tables with ease
- **Column Designer** - Define columns with data types, primary keys, and constraints
- **Visual Relationships** - Create and visualize relationships between tables
- **Relationship Types** - Support for various relationship cardinalities (one-to-one, one-to-many, many-to-many)
- **Connection Points** - Visual connection points on tables for creating relationships

### Canvas & Workspace
- **Interactive Canvas** - Large workspace area for designing your database schema
- **Pan & Zoom** - Navigate large diagrams by panning (right-click or middle-mouse drag)
- **Minimap** - Bird's-eye view of your entire diagram for quick navigation
- **Click-to-Navigate** - Click on minimap to quickly jump to different areas
- **Auto-scroll** - Automatic scrolling when dragging elements near canvas edges
- **Grid Layout** - Organized arrangement of database elements

### Annotations & Documentation
- **Sticky Memos** - Add sticky notes and annotations to your diagrams
- **Visual Notes** - Document important information directly on the canvas
- **Color-coded Elements** - Visual distinction between different types of elements

### View Controls
- **Visibility Toggles** - Show/hide tables, relationships, memos, and minimap independently
- **Table List** - Sidebar panel showing all tables in the current database
- **Focus Mode** - Click on table names in the sidebar to focus on specific tables
- **Tab Navigation** - Switch between Tables and View panels for better organization

### User Experience
- **Responsive Design** - Works seamlessly across different screen sizes and devices
- **Interactive Tutorial** - Built-in intro.js guided tour for new users
- **Context-sensitive Help** - Tutorial steps that explain each feature in detail
- **Empty States** - Helpful prompts when starting with a blank canvas
- **Fast & Easy** - Quick and intuitive interface for rapid database design

### Feedback System
- **User Feedback Form** - Built-in feedback mechanism for users to share suggestions
- **Multiple Contact Channels** - GitHub, Facebook, and website links for reaching developers
- **Feedback Prompts** - Periodic prompts to gather user feedback (1-hour intervals)
- **Netlify Forms Integration** - Secure form submission for feedback

### Visual Design
- **Modern UI** - Clean and professional interface design
- **SVG Icons** - Crisp vector icons throughout the application
- **Color Scheme** - Consistent color palette for a cohesive look
- **Smooth Animations** - Polished transitions and hover effects
- **Hero Section** - Welcoming landing page with feature highlights

### Technical Features
- **Browser-based** - No installation required, runs entirely in the browser
- **No Backend Required** - Fully client-side application
- **localStorage API** - Persistent data storage without server dependencies
- **Spam Protection** - Honeypot field for form security
- **Cross-browser Compatible** - Works on modern browsers

### Navigation
- **Multi-page Structure** - Separate pages for home, workspace, about, and feedback
- **Breadcrumb Navigation** - Easy back-to-home navigation from any page
- **Persistent State** - Database state persists across page visits
- **Deep Linking** - Direct access to specific databases via URL parameters

## About Page Features

- **Feature Overview** - Visual grid showcasing all main features
- **Social Links** - GitHub and other social media connections
- **Branding** - Logo and tagline display
